# Berserkr VTT System

Foundry VTT System for the **Berserkr** RPG (based on Mörk Borg).

## Technical Stack
- **Foundry VTT API**: v13 (Compatible with v12)
- **Language**: TypeScript
- **Frontend**: Svelte
- **Build Tool**: Vite

## Rules Summary
See `docs/rules-reference/` for detailed game mechanics.

## Setup for Developers
1. Clone this repo into `Data/systems/berserkr`.
2. Run `npm install` (inside docker).
3. Run `npm run build` to compile.
