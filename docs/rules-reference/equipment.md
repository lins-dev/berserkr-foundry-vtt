# Equipment

O equipamento é medido em **Trinkets (t)**, a moeda comum. A capacidade de carga é determinada pelo atributo **Might**.

## Capacidade de Carga
- Pode carregar **Might⁺⁸** itens de tamanho normal sem problemas.
- Acima disso, todos os testes de Might e Swift sofrem **+2 de DR**.
- É impossível carregar mais que o dobro de Might⁺⁸.

Itens de tamanho normal incluem ossos, cabeças, machados, etc. Objetos pesados (baús, bestas, inimigos abatidos) contam como vários itens ou exigem meios especiais de transporte.
