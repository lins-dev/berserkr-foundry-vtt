# Core Rules Summary - Berserkr

## Abilities (Attributes)
Berserkr utiliza 5 atributos base (diferente dos 4 do Mörk Borg):
1. **Fortitude**: Parry, survive falling, resist poison/cold/heat.
2. **Might**: Melee attack, lift, crush, move objects.
3. **Guile**: Ranged attack, perceive, charm, stealth.
4. **Swift**: Defend, balance, swim, climb.
5. **Wits**: Wield runes, fix stuff, find your way, remember.

### Ability Score Generation
- Roll **3d6** + class modifiers.
- Use table to determine modifier:
  - 1-4: -3
  - 5-6: -2
  - 7-8: -1
  - 9-12: ±0
  - 13-14: +1
  - 15-16: +2
  - 17-20: +3
  - 21-23: +4

## Core Mechanics
- **Tests**: Roll **d20 ± Ability** vs **Difficulty Rating (DR)**.
- **Difficulty Ratings (DR)**:
  - DR6: Simple
  - DR8: Routine
  - DR10: Pretty Simple
  - DR12: Normal
  - DR14: Difficult
  - DR16: Really Hard
  - DR18: Nigh Impossible

## Combat
- **Initiative**: 
  - Individual: Test **Swift + d6**.
  - Group: Roll **d6** (1-3 Enemies first, 4-6 PCs first).
- **Attacking (Melee)**: Test **Might DR12**.
- **Attacking (Ranged)**: Test **Guile DR12**.
- **Defending**: Test **Swift DR12**.
- **Parry**: Test **Fortitude DR+3** (Once per battle). Success: Negate attack + deal weapon damage immediately.
- **Critical Hit (Nat 20)**: Double damage, target armor -1 tier.
- **Fumble (Nat 1)**: Weapon breaks/lost (Attack) or x2 damage + armor -1 tier (Defense).

## Health & Recovery
- **Hit Points (HP)**: Fortitude + d10 (if classless).
  - 0 HP: Broken.
  - Negative HP: Dead.
- **Rest**:
  - Short: d4 HP.
  - Long: d6 HP + regain Fates + regain Rune uses.

## Fates (Luck)
- Begin with **d2 Fates**.
- Uses: Max damage, reroll, reduce damage by d6, negate crit/fumble, DR -4.
